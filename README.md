## Rodar o projeto:

- git clone https://github.com/larissaluanaleal/challenge-bills.git
- Execute `npm install`
- Execute `json-server --watch data-base.json` deixe rodando.
- Em outro terminal execute `ng serve -o`
- Acesse http://localhost:4200/
